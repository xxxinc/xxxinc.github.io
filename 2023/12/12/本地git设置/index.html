经过调试，在我的电脑上需要进行这样的设置：（太麻烦了）

  1. cmd 输入：

    git config --global --unset http.proxy

    git config --global --unset https.proxy

  2. VPN设置为 自动配置系统代理 和 全局模式

  3. 电脑 设置 => 网络与Internet => 代理 , 打开 自动检测设置 和 使用代理服务器

具体原因没有深究，应该是本地代理地址和VPN代理地址之间出现的一些矛盾，我的想法是完全设置为VPN代理来解决（玄学）。
